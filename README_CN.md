## 主要发现

1. 认知水平和能力自信是最强预测因子 (r = 0.448, 0.423)
2. 学历悖论：低学历父亲参与度反而更高
3. 中介效应：能力自信中介42.5%的认知-参与关系
4. 夫妻一致性 (r = 0.267) 影响是配偶态度 (r = 0.119) 的2.2倍

- 开放性问题回答

## 分析方法

1. 描述性统计：均值、标准差、频率
2. 相关分析：皮尔逊相关系数及95%置信区间
3. 回归分析：多元线性回归、标准化系数
4. 中介效应分析：Baron and Kenny方法
5. 效应量分析：Cohen's d、R平方
